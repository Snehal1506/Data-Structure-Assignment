Map Data Structure

A Map is a collection of elements where each element is stored as a Key-Value pair. Map object can hold both objects and primitive values as either key or value, any value can be used as a key , even objects.

It stores the key-value pairs in the insertion order. i.e. When we iterate over the map object it returns the key, value pair in the same order as inserted.

Why we need Map if we have Object s

  1. A Map is similar to Object, but any value can be used as key in maps, but keys in objects are only Strings and Symbols
  2. The key—value pairs in Map maintains the insertion order, whereas Object don’t.
  3. You can get the size of the Map, but we don’t have a built-in method to get size in Object (but we can do it manually).
  4. Maps are iterable, whereas an Object is not iterable by-default.
  5. Maps have additional methods not available to normal objects

Syntax to create map: 

        new Map([it])

        Parameter:
        
        it - It is any iterable object whose values are stored as 
             key, value pair,
             If the parameter is not specified then a new map is created 
             is Empty
        
        Returns:
        
        A new Map object

Now lets create Map using the Map object

        
            //Creating a Map

            var map1 = new Map([["firstname", "Snehal"], ["lastname", "Lokhande"], ["class", "BE"]]);

            map1;//{"firstname" => "Snehal", "lastname" => "Lokhande", "class" => "BE"}

Properties: 

        Map.prototype.size - It returns the number of elements or the key-value pairs in the map.
    
        
            //Size

            map1.size; //3

Methods:

  1. Map.prototype.set() - It adds the key and value to the Map Object.
    Syntax: 
                    map1.set(k, v);
    
                Parameter:
                
                k - Key of the element to be added to the Map
                v - value of the element to be added to the Map
                
                Returns:
                
                It returns a Map object
            
                    
                    //set(key , value) method
        
                    map1.set("Branch","IT");

                    map1; // {"firstname" => "Snehal", "lastname" => "Lokhande", "class" => "BE", "Branch" => "IT"}
  2. Map.prototype.has() - It return a boolean value depending on whether the specified key is present or not
    Syntax: 
                    map1.has(k);
    
                Parameter:
                
                k - Key of the element to checked
                
                Returns:
                
                true if the element with the specified key is present 
                or else returns false.
            
                    
                    //has(key) method
        
                    map1.has("firstname"); // true

                    map1.has("age"); //false
  3. Map.prototype.get() - It returns the value of the corresponding key
    Syntax: 
                    map1.get(k);
    
                Parameter:
                
                k - Key, whose value is to be returned
                
                Returns:
                
                The value associated with the key, if it is present 
                in Map, otherwise returns undefined
            
                    
                    //has(key) method
        
                    map1.get("firstname"); // Snehal

                    map1.get("age"); // undefined
  4. Map.prototype.delete() - It delete’s both the key as well as a value from the map.
    Syntax: 
                    map1.delete(k);
    
                Parameter:
                
                k - Key which is to be deleted from the map 
                
                Returns:
                
                true if the value is found and deleted from 
                the map otherwise, it returns false
            
                    
                    //delete(key) method
        
                    map1.delete("firstname"); // true

                    map1.delete("age"); // false
  5. Map.prototype.clear() - Removes all the elements from the Map object
    Syntax: 
                    map1.clear();
    
                Parameter:
                
                No parameters
                
                Returns:
                
                undefined
            
                    
                    //clear method
        
                    map1.clear(); // undefined

                    map1; // Map(0) {}
  6. Map.prototype.entries() - It returns an iterator object that contains key/value pair for each element present in the Map object.
    Syntax: 
                    map1.entries();
    
                Parameter:
                
                No parameters
                
                Returns:
                
                It returns an iterator object
            
                    
                    //entries --> returns key value pair
        
                    map1.entries(); // MapIterator {"firstname" => "Snehal", "lastname" => "Lokhande", "class" => "BE"}
  7. Map.prototype.keys() - It returns an iterator object which contains all the keys present in the Map Object.
    Syntax: 
                    map1.keys();
    
                Parameter:
                
                No parameters
                
                Returns:
                
                An iterator object
            
                    
                    //keys
        
                    map1.keys(); // MapIterator {"firstname", "lastname", "class"}
  8. Map.prototype.values() - It returns an iterator object which contains all the values present in the Map Object.
    Syntax: 
                    map1.values();
    
                Parameter:
                
                No parameters
                
                Returns:
                
                An iterator object
            
                    
                    //values
        
                    map1.values(); // MapIterator {"Snehal", "Lokhande", "BE"}
  9. Map.prototype.forEach() - It executes the callback function once for each key/value pair in the Map, in the insertion order.
    Syntax: 
                    map1.forEach(callback[, thisArgument]);
    
                Parameter:
                
                callback - It is a function which is to be executed for each element of the Map.
                thisargument - Value to be used as this when executing the callback.
                
                Returns:
                
                undefined

    The callback function is provided with three parameters as follows:

    • the element key
    • the element value
    • the Map object to be traversed

                    
                    //Using forEach
        
                    // creating a map 
                    var map1 = new Map([["firstname", "Snehal"], ["lastname", "Lokhande"], ["class", "BE"]]);
                            --------------------------------------------------------------------
                    Declaring a call back function we are using only one parameter value so it will ignore other two . 
                    function printOne(values) {
                        console.log(values);
                    }
                    console.log("-----one parameter-----");
                    map1.forEach(printOne);
                    //-----one parameter-----
                    //Snehal
                    //Lokhande
                    //BE
                            --------------------------------------------------------------------
                    Declaring a call back function we are using two parameter value so it will ignore last one
                    function printTwo(values, key) {
                        console.log(key + "  " + values);
                    }
                    console.log("-----two parameter-----");
                    map1.forEach(printTwo);
                    //-----two parameter-----
                    //firstname  Snehal
                    //lastname  Lokhande
                    //class  BE
  10. Map.prototype[@@iterator]() - It returns an Map iterator function which is entries() method of Map object by default.
    Syntax: 
                    map1[Symbol.iterator]
    
                Parameter:
                
                No parameters
                
                Returns:
                
                Returns an map iterator object and it is 
                entries() by default
            
                    
                    //Symbol.iterator
        
                    var getit = map1[Symbol.iterator](); // MapIterator {"firstname" => "Snehal", "lastname" => "Lokhande", "class" => "BE"}

                    for (var elem of getit)
                    console.log(elem); 
                    //it prints
                    //["firstname", "Snehal"]
                    //["lastname", "Lokhande"]
                    //["class", "BE"]
